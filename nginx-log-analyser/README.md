# Log Archive Tool
A bash script to archive logs.  

## Getting Started
1. **Clone the repository**
    ```
    git clone git@github.com:dAnkc0der/DevOps-Projects.git
    cd nginx-log-analyser
    ```
2. **Make the script executable**
    ```
    chmod +x log-analyser.sh
    ```
3. **Execute the script**  
    ```
    ./log-analyser.sh
