# Log Archive Tool
A bash script to archive logs.  

## Getting Started
1. **Clone the repository**
    ```
    git clone git@github.com:dAnkc0der/DevOps-Projects.git
    cd log-archive
    ```

2. **Perform below steps**
    ```
    mv log-archive.sh archive
    chmod +x archive
    mv archive /usr/local/bin
    ```

3. **Execute the Script**
    ```
    archive /var/log
    ```
    This project is a part of [roadmap.sh](https://roadmap.sh/projects/log-archive-tool)
